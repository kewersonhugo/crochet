Crochet is a layered language for generative narratives.

Logical language:

  relation Sig.
  actor Name :: Role*.
  when Pred { Behaviour }.
  [repeatable] action Name when Pred { Behaviour }.
  scene Name { Behaviour }.
  context Name { when/action/scene* }.

Functional/imperative language:

  command Sig { Behaviour }.
  do { Behaviour }.


  let X = Expr.

  match { when*, else }.
  invoke.

  goto Scene.
  fact Pred.
  forget Pred.
  enable [only] Context.
  disable Context.
